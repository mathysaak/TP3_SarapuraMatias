Nombre Descripción Ejemplo
if Permite ejecutar un bloque de código si se cumple una condición if (condicion) {
// Bloque de código a ejecutar si la condición es verdadera
}
else Define un bloque de código a ejecutar si la condición de un if no se cumple if (condicion) {
// Bloque de código a ejecutar si la condición es verdadera
} else {
// Bloque de código a ejecutar si la condición es falsa
}
for Itera sobre una secuencia de elementos un número específico de veces for (inicialización; condición; actualización) {
// Bloque de código a ejecutar en cada iteración
}
